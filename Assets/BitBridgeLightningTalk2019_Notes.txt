Taken from ProBuilder 
	Use Unity Package Manager to install ProBuilder. 
	Under Packages in your hierarchy navigate to ProBuilder/Content/Material/Textures/Checker
	Copy and Paste to your projects texture art folder.

basic_soldier_red created with GIMP
	1. Import the basic_soldier_blue texture into GIMP
	2. Select All (Ctrl+A) 
	3. Under the Colors Tab -> Map -> Rotate Colors
	4. Use the Source Range and Destination Range Color Wheels to change the blue uniform color to any color you like.
	5. Under the File Tab select Export As... to export your new basic_soldier_red
